# 北京市浅层滑坡易发性评价信息系统 - 静态网站原型

## 使用方式
直接双击 `index.html` 即可在浏览器中打开。

## 功能范围
- 登录首页：角色切换、登录模拟、快捷入口跳转。
- 数字大厅：KPI、易发性空间分布、流程、因子贡献、趋势图。
- 易发性地图：图层筛选、空间查询、地图弹窗、区域详情、结果表格。
- 模型评价：标签页、运行记录、ROC/PR-AUC/SHAP 可视化。
- 数据管理：数据入库模拟与数据资产目录。
- 报告导出：报告预览与导出按钮交互。

## 注意
本网站为课程作业 UI 原型，不连接数据库，不提供真实 GIS 服务或模型 API。页面中的数值为演示值，正式论文中应以最终复核结果替换。
